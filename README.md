# Huawei Note Templates Collection

本仓库收集了适用于华为笔记（Huawei Notes）的一组实用模板，方便日常记录、计划与思考整理。

## 仓库内容

- `1.html`：模板示例页面 1
- `2.html`：模板示例页面 2
- `华为笔记模板.jpeg`：模板预览图像
- `华为笔记模板.pdf`：模板 PDF 文件
- `华为笔记模板.ppt`：模板演示文档
- `简约三段式日常计划 - 对齐优化版.pdf`：简约日常计划模板
- `简约三段式日常计划 - 对齐优化版_cropped.pdf`：裁剪版简约日常计划模板

## 使用说明

1. 打开 HTML 文件查看模板布局。
2. 根据需要将内容复制到华为笔记中。
3. 如果需要打印或分享，可使用 PDF 文件。

## 适用场景

- 日常生活计划
- 工作任务记录
- 学习笔记整理
- 会议纪要与复盘

## 目标

为用户提供一套清晰、简洁、易复制的华为笔记模板，帮助更高效地记录和整理信息。
